Delayota adalah website pembelajaran sederhana yang dibuat untuk belajar dasar pembuatan website menggunakan HTML tanpa CSS dan tanpa layout.

Website ini bertujuan untuk memahami struktur HTML, penggunaan tag, dan cara membuat halaman web yang rapi dan valid.

Dengan konsep sederhana, Delayota cocok untuk pemula yang baru mulai belajar dunia web development.
